Diving deeper in Week Two

When deep calls deep, it's time to dive deeper for sure. Week two involved delving into what we had started in week one.

We further went deeper into git in the cloud by looking at online version control platforms that enabled developers to collaborate on projects. We focussed more on Github. We further learnt about Git branches, pull requests and merging in git.

We further covered more CSS properties. We got introduced to Bootstrap a CSS framework that is used popularly by developers to build mobile-responsive web pages.

We further covered the Bootstrap Grid system and how it's used to place items correctly in the page. We covered different classes used in Bootstrap. We closed the week by building A personal Blog site and hosted it on Github pages.
