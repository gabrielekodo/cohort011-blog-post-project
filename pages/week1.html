My First 7 days of Software Engineering

It's often said that a jorney of a thousand miles begins with a step. Here is my story.

The Software Engineering Program at Refactory is a very fast-paced plan. Each single has a specific topic to be covered. My very first lecture was an introduction to how the Internet works where I was introduced to concepts such as computer networking, servers, domain names, IP Addresses, TCP/IP protocols.

Day Two was mainly about Software design processes and steps which includes Budgetting, creating a team,choosing the tools to be used for the project e.g project management tools, the programming language, frameworks etc, the development process(e.g agile development).

Day Three was introduction to version control systems. We were introduced to Git and Github. I learnt about the different commands used in git CLI such git add, git commit, git push, git remote, git pull I learnt how to create a repository locally and remotely. I learnt how to create an online repository on Github and both push and pull to and from it.

On the fourth day I was introduced to HTML(a markup language used to build websites). I learnt about different element selectors used for varying purposes.

The fifth day was an introduction to CSS. CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once.
